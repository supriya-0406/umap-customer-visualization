# 📊 Customer Visualization using UMAP

## 📌 Overview

This project uses UMAP to reduce high-dimensional customer data into 2D for visualization and pattern discovery.

---

## 🎯 Objective

Understand customer behavior using dimensionality reduction.

---

## 🧠 Model

UMAP (Uniform Manifold Approximation and Projection)

---

## ⚙️ Technologies

* Python
* Pandas
* Scikit-learn
* UMAP
* Matplotlib

---

## 🚀 How to Run

pip install -r requirements.txt
python main.py

---

## 📈 Output

* 2D visualization of customers
* Cluster-based grouping

---

## 💡 Business Impact

* Customer segmentation
* Pattern discovery
* Better targeting strategies

---

## 🎯 Learning Outcomes

* Dimensionality reduction
* Data visualization
* Clustering integration
